## 0. 设计原则速览（6 条决策）

| # | 维度 | 决策 |
|---|---|---|
| 1 | 顶层选项预算 | **3-4 个**，按场景灵活分类，不强制统一模板 |
| 2 | 新增内容范围 | NPC flavor pockets + 桥接喘息夜 2 个 |
| 3 | 时间成本 | 喘息夜全免费；日间进 NPC 对话付 1 tick，子话题免费 |
| 4 | 重构范围 | 选项 ≥5 的场景（共 9 个）；4 选项保持扁平；boss 战扁平不变 |
| 5 | 喘息夜形态 | 夜 1：线性短章（4 人各冒一句心里话）；夜 2：选 1-2 人深聊，轻度影响 boss 支援 |
| 6 | flavor 门槛 | 阶段门槛（早/中/晚），看过即消失，喘息夜 2 解锁晚期深聊 |

## 1. 渐进式选择 — 通用规则

### 1.1 二级菜单 ink 写法

用 `= 子菜单_xxx` stitch（同 knot 下），不是 nested choice。

ink 模板：

```ink
=== 场景_xxx ===
= 顶层
  [场景描述]
  * [大方向 A] -> 子菜单_A
  * [大方向 B] -> 子菜单_B
  + 推进选项（视状态） -> 下一场景

= 子菜单_A
  [一句过渡氛围]
  * 〔技能 XX〕动作 1 -> 动作_1
  * 动作 2 -> 动作_2
  + [← 还是先做点别的] -> 顶层

= 子菜单_B
  ...
```

### 1.2 时间成本规则

- 顶层 → 二级：**免费**（无 tick）
- 二级 → 真实动作：**消耗 tick**，动作完成后 `-> 顶层`（不回二级）
- 二级 → 顶层（"还是先做点别的"）：**免费**

设计目标：**浏览不要钱，落子才计费**。玩家可以放心点进二级"看看有什么"再回退。

### 1.3 选项数限制

- 顶层：**3 个**为常态；**4 个**为偶尔（如对话菜单含 4 NPC）
- 二级：上限 **3-4 个**（含返回按钮则 4-5）
- 单一选项可不进二级，直接 divert 到事件 knot

### 1.4 文案规范

- 顶层选项前缀用方括号：`[调查寝室]`
- 二级菜单首行加一句过渡氛围（"你环顾四周，决定从哪儿入手……"）
- 返回按钮统一文案：`← 还是先做点别的`（可在二级菜单的不同语境微调，如"我再想想"）
- 〔技能 XX〕标记保留在二级（动作粒度）

## 3. 喘息夜 1（Act 1 → Act 2 之间）

### 形态：A — 线性短章 / 集体氛围片段

- 位置：Act 1 终幕之后、Act 2 启幕之前
- 长度：~100 行 ink，约 5-8 屏
- **全程免费**（不消耗 tick）
- 玩家互动度：低（偶尔 1-2 个无后果的氛围选择）

### 大纲（6 段，Y 路线 3 NPC + PC 内心独白）

1. **场记开头**：深夜 2 点，第一波恐怖结束。302 室，铁皮缝里漏进路灯。每个人各占一角。一句环境描写。
2. **PC 内心独白**（高三压力主题，原"昕开口"段改为 PC 第二人称内心戏）：你想到明天数学综合考的事，张了张嘴又咽回去，没人开口。
3. **志勇插一句**（想吃的东西主题）：他说他妈昨天问他想吃什么。语气故意像没事人。
4. **沉默**（玩家选择："你先开口" / "等他们先" — 无后果，影响下一段叙述视角而已）
5. **张怡的一句话**（对未来的冷感主题）：他没回头，望着窗，说一句让人愣住的话（"如果出不去，可能也行"），其他人不知该接什么。
6. **砚波收尾**（你和他的旧事主题）：他凑过来，提起一件你们高一一起干的事，让你们俩都笑了一下。然后他熄灯。
7. **过场到 Act 2**

### 设计意图

- 不让玩家做决定，给纯抒情时刻
- 3 NPC（志勇/张怡/砚波）各冒一句心里话 + PC 自身内心独白 = 4 段心理写实
- 张怡那句"如果出不去也行"是后期他"对死亡的看法"的伏笔

## 4. 喘息夜 2（Act 2 → Act 3 之间）

### 形态：B — 角色聚焦深聊 + 轻度影响 boss

- 位置：Act 2 终幕之后、Act 3 启幕之前
- 长度：~200-280 行 ink（共享外壳 ~60 行 + **3 套**深聊 × 50-70 行）
- **全程免费**

### 结构骨架（Y 路线 3 NPC）

```ink
=== 喘息夜2 ===
= 入夜
  [氛围铺垫：明天就要冲了。今晚是最后一夜。]
  -> 选谁

= 选谁
  你想多陪谁一会儿？
  * 多陪志勇一会儿 -> 深聊_志勇
  * 多陪张怡一会儿 -> 深聊_张怡
  * 多陪砚波一会儿 -> 深聊_砚波

= 深聊_志勇
  [40-60 行内容]
  ~ 陪伴_志勇 = true   // 用于 boss 战支援判定
  -> 第二轮选谁

= 第二轮选谁
  夜更深了。还有时间，再陪一个人？
  + {not 陪伴_志勇} 多陪志勇 -> 深聊_志勇
  + {not 陪伴_张怡} 多陪张怡 -> 深聊_张怡
  + {not 陪伴_砚波} 多陪砚波 -> 深聊_砚波
  + 不陪了，闭眼睡 -> 天亮

= 天亮
  [过场到 Act 3]
```

### 玩家路径

- 一夜可深聊 1-2 人（不能 3 人全聊——主题"陪伴是稀缺品"）
- 选了的人会在 boss 战阶段 2 多一个支援选项（如 `{陪伴_志勇} * 志勇主动挡刀 -> 支援_志勇`）
- 不强制每个 NPC 都有支援选项 → boss 战避免变成"看你陪了谁"的检定

### 设计意图

- "选谁陪最后一夜" = 高三+末日双重设定下最有戏剧性的决断
- 玩家有具体反馈（boss 战可见）但不分支结局——保住可控性
- 3 套深聊承载 NPC 晚期 flavor 主题（详 §5）
- PC 自身内心戏不开 flavor pocket，由 SAN 系统（理智_昕 已被消耗到低位时的 # state:san_xin tag + 强制 E5 触发）承担恐怖代价

- **晚期**：仅在喘息夜 2 解锁，且每人只能选 1 个（不能"陪一个人"把 3 段全看完——稀缺性）

### Phase 门槛

- 早期：`{act_phase >= 1}` 全程可见
- 中期：`{act_phase >= 2}` Act 2 起出现
- 晚期：`{进入喘息夜2}` 限定

### 主题白名单 / 黑名单

- ✅ 允许：高三压力、想吃的东西、对未来的恐惧、童年小事、爱情/暗恋、和家人的关系、厨师梦、对死亡的看法

## 7. 风险与未决项

- **a3_共识抉择**：现版本可能已经有"听 NPC 立场"机制；plan 阶段需读全场景再确定如何重构，避免重复
