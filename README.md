# 控江隔离 · 序章 Ink Demo

[**▶ 在浏览器里玩**](https://luoxin9510.github.io/kongjiang-ink-demo/)（移动端友好）

CoC 7e 跑团模组「控江隔离」的 Act 0 序章 — 用 [Ink](https://github.com/inkle/ink) 写的可玩分支脚本。

## 设定

2022 年 4 月，上海，控江路三号楼 302 室。隔离第七天的傍晚。
四个人挤在 15 ㎡ 的男生宿舍里：昕、砚波、志勇、张怡。
今天和过去六天没有不同 —— 直到现在还没有不同。

## 玩法

- 点击选项推进剧情
- 顶部 HUD 显示 PC 理智 / 三位队友的恐惧值
- 状态变化通过 Ink 变量驱动
- 主线大约 10–15 分钟

## 技术

- **Ink 脚本**（中文 identifier）→ `inklecate v1.2.0` 编译为 JSON
- **inkjs 2.3.1** 浏览器 runtime
- 单文件 HTML，无构建依赖
- 暗色主题、中文 serif、iPhone Safari 适配

## 源码

完整 Ink 源码（含 helpers 与设计文档）位于主仓库 `trpg-bot/worlds/modules/kongjiang-isolation/ink_demo/`。
